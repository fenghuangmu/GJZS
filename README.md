## 构建

- 在此之前您需要自行配置好 Java 与环境变量，推荐使用[IntelliJ IDEA](https://www.jetbrains.com/zh-cn/idea/)

```bash
git clone https://github.com/liuran001/GJZS.git
cd GJZS
./gradlew :app:assembleRelease
```
